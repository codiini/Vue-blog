# blog-tutorial

### This is a small blog created using Vuejs and Firebase as the backend, nothing fancy
## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
